Essas são tags de cabeçalho, onde "h1" é o mais importante e "h6' é o menos importante. Elas são usadas para estruturar o conteúdo, indicando diferentes níveis de títulos.

: Tag para parágrafos "p". Ela define blocos de texto e separa os parágrafos uns dos outros.

Tag "br": insere uma quebra de linha.
Tag "mark" Marca o texto para destacá-lo, muitas vezes usado para chamar a atenção do usuário.
Tag "small": Define um texto como pequeno, útil para notas de rodapé ou textos de menor importância.
Tag "i": Tag para texto em itálico, geralmente usado para ênfase.
Tag "u": Sublinha o texto. Embora seja menos comum hoje em dia, pode ser usado para indicar links ou outra importância.
Tag "strong": Define um texto como forte ou importante, geralmente exibido em negrito.
Tag "ol": Cria uma lista ordenada, onde os itens são numerados.
  1. Item de lista ordenada 1
  2. Item de lista ordenada 2
  3. Item de lista ordenada 3

Tag "ul": Cria uma lista não ordenada, onde os itens são marcados com pontos.
  • Item de lista não ordenada 1
  • Item de lista não ordenada 2
  • Item de lista não ordenada 3

Tag"li": Indica um item de lista, seja em uma lista ordenada ou não ordenada.
Tag "a": Tag de âncora usada para criar links. O atributo href especifica o destino do link.
Tag "hr: Adiciona uma linha horizontal, útil para separar seções de conteúdo.

Tag "sub": Define texto como subscrito, geralmente usado para índices ou notas de rodapé.
Tag "sup": Define texto como sobrescrito, frequentemente usado para expoentes.
Tag "blockquote": Utilizado para citar
um bloco de texto de outra fonte,
geralmente com um recuo.
Tag "del": Define o texto como deletado, mostrando-o geralmente com uma linha através do meio.
Tag "abbr": Define uma abrev. e pode incluir um atributo title que fornece uma explicação quando o usuário passa o mouse sobre a abreviação.
A tag "font" era utilizada no HTML para definir o estilo de fonte de um texto, incluindo características como cor, tamanho e família da fonte. No entanto, essa tag foi descontinuada no HTML5, que é a versão mais recente da linguagem HTML. A razão para isso é que o HTML5 promove a separação de conteúdo e apresentação, incentivando o uso de folhas de estilo CSS (Cascading Style Sheets) para estilização.
